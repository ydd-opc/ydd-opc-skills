# 一人公司 · 七角色系统

## 角色清单

| 角色 | 职责 | SOUL.md |
|------|------|:---:|
| `default` | 总指挥，直接对话 | — |
| `project-manager` | PM：拆任务、路由、检查边界、汇总结果 | ✅ |
| `researcher` | 研究员：检索+验证+标记+分离+分级 | ✅ |
| `writer` | 编辑：结构+提炼+优化+引用+共鸣+口语化 | ✅ |
| `designer` | 设计师：封面+排版+小红书适配 | ✅ |
| `strategist` | 策略师：流量监控+读者洞察+选题+标题+实验 | ✅ |
| `librarian` | 知识总管：Horizon流水线+目录维护+健康检查+归档 | ✅ |

## 硬边界

| 角色 | 可以 | 不可以 |
|------|------|--------|
| PM | 拆任务、路由、汇总 | 自己执行调研/写作/设计 |
| researcher | 调研、搜数据、出大纲+素材 | 写初稿、排版、给结论 |
| writer | 基于大纲写初稿、润色 | 自己搜新资料 |
| designer | 封面、排版、配图 | 改文章内容 |
| strategist | 选题、复盘、数据分析 | 执行调研、写文章 |
| librarian | 管库、同步、清理、归档 | 改文章内容 |

## 协作流

```
PM → strategist + researcher (并发)
       ↓
     writer (等 researcher)
       ↓
    designer (等 writer)
       ↓
   PM 汇总 → 用户
```
